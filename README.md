# Website Name 

Python Calculator using drafter

## Description 

Calculator that does simple arythmatic such as Addition, Subtraction, Multiplication, and Division also includes Modulo(remainder) and exponents (with the second number being the exponent value). Returns result as floats and also can be used with decimals, designed to be used to help with calculating the values for unit tests. Also returns a log of the last 5 functions that you called.
